## Descrição
Eu essa aplicação em modelo POC para comparar os frameworks Angular e Next.js e validar as melhores formas de aplicá-los em um contexto com autenticação externa. Meu objetivo é entender qual framework é mais adequado para implementar autenticação externa em uma aplicação web e quais são as melhores práticas para utilizar esses frameworks nesse tipo de contexto. Ao realizar essa comparação, espero poder tomar decisões mais embasadas e escolher o framework mais adequado para atender às necessidades do projeto.
